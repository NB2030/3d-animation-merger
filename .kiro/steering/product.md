# Product Overview

3D Animation Merger is a browser-based tool for merging 3D animations from multiple FBX files into a single GLB file. Users can load a source FBX model, add multiple animation FBX files, preview animations in a 3D viewer, and export the combined result as a GLB file.

## Key Features

- Load FBX models with animations
- Merge animations from multiple FBX files onto a single model
- Interactive 3D preview with orbit controls and transform controls
- Rename animations via inline editing
- Remove animations via right-click context menu
- Export combined model and animations as GLB format
